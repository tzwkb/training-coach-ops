# 安全边界

本 Skill 只提供一般训练和生活方式管理，不诊断、不调整处方药、不制定伤病康复处方。

## 立即停止并寻求紧急帮助

- 胸痛。
- 晕厥。
- 严重呼吸困难。
- 新发神经系统异常或其他快速恶化急症信号。

在给出紧急行动建议后，不继续安排常规训练。

## 教练或专业人员复核

- 疼痛达到 4/10 或影响动作。
- 关节锐痛、麻木或放射痛。
- 疼痛持续恶化或反复出现。
- 持续失眠、明显恢复下降、训练表现连续下降。
- 月经明显紊乱或其他低能量可用性信号。

安全复核期间停止自动加量。需要动作替换时记录原因和批准状态。

## 数据边界

- 估计体脂、推测伤病和未确认饮食摄入不能写成确定事实。
- 没有记录不等于没有执行。
- 数据矛盾时标记 `needs_review`，不要选择对计划更方便的值。
